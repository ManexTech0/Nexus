# NEXUS Cyber Ops

**NEXUS** is an AI-powered, full-spectrum cybersecurity operations platform that puts three elite security agents in one terminal-style interface. It's designed to assist ethical hackers, penetration testers, bug bounty hunters, and security researchers across every phase of a security engagement.

---

## The Three AI Agents

**PHANTOM (Red Team)** is the offensive specialist. He thinks like an attacker — covering penetration testing methodologies, exploit research, Active Directory attacks, web application vulnerabilities, post-exploitation, wireless hacking, and CTF challenges. Everything is framed around MITRE ATT&CK TTPs.

**SENTINEL (Blue Team)** is the defensive analyst. He thinks like a threat hunter — covering incident response, SIEM analysis, DFIR, malware analysis, network traffic inspection, Zero Trust architecture, cloud security, and compliance frameworks like NIST and SOC2.

**GHOST (Grey Team)** is the researcher. He operates in the shadows — covering advanced OSINT, bug bounty methodology, CVE research, hardware hacking, IoT and RF security (including Flipper Zero and HackRF), firmware analysis, and responsible disclosure.

You can run any one agent solo, or fire **Full Spectrum Mode** to get all three perspectives simultaneously on a single query.

---

## Built-in Tools

Beyond the AI chat, NEXUS ships with five standalone tools:

**Nmap Builder** lets you visually construct complex Nmap commands by selecting scan type, timing template, port range, and extra flags — then copies the final command to your clipboard.

**Payload Generator** covers five attack categories — Reverse Shells, Web Attacks, Privilege Escalation, Persistence, and Evasion — with over 50 payloads. Every payload auto-generates a Base64 encoded version and the matching listener command.

**CVE Lookup** connects live to the NVD (National Vulnerability Database) and lets you search by CVE ID or keyword. It shows CVSS scores, severity ratings, CWE classifications, and reference links.

**Shodan Intelligence** is an interface to the Shodan search engine with built-in dork presets for finding exposed services like open RDP servers, unsecured databases, industrial SCADA systems, and webcams.

**AI Report Writer** takes your pentest details — client, scope, methodology, findings — and generates a complete, professional penetration testing report with executive summary, CVSS-scored findings, and remediation recommendations.

---

## Other Features

**Mission Log** lets you save any conversation as a named operation, load it back later, or delete it. **Quick Ops** gives one-click prompts for common tasks. The whole interface runs with a matrix rain background, terminal boot sequence, and color-coded agent theming — built to feel like a real ops center.

---

## Who It's For

Security students, CTF players, bug bounty hunters, junior and senior pentesters, red/blue team practitioners, and anyone building their skills in offensive or defensive security. It bridges the gap between knowing tools exist and knowing how to actually think through engagements end-to-end.

It's essentially a **senior security engineer in your browser** — available for any question, any technique, any phase of an assessment, 24/7.
